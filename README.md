# SubStrings - Поиск подстроки

Учебный проект по реализации и сравнению алгоритмов поиска подстроки на C++.

## 📚 Описание

В реализованы четыре алгоритма поиска подстроки:

| Алгоритм | Файл | Суть |
|----------|------|------|
| **Полный перебор** | `textScan.cpp` | Последовательная проверка всех позиций |
| **КМП (Кнут-Моррис-Пратт)** | `textScan.cpp` | Сдвиги по префиксу шаблона |
| **Бойер-Мур-Хорспул** | `textScan.cpp` | Сдвиги по суффиксу текста |
| **Бойер-Мур (полный)** | `textScan.cpp` | Плохой символ + хороший суффикс |


## 📁 Структура проекта

```
src/
├── main.cpp          # Точка входа, запуск тестов
├── test.cpp          # Функции тестирования и замера времени
└── textScan.cpp      # Реализация алгоритмов поиска

headers/
└── textScan.h        # Заголовочный файл с декларациями

report/
├── output.txt        # Полный вывод программы
└── report.md         # Отчет с анализом результатов

CMakeLists.txt        # Конфигурация сборки
README.md             # Этот файл
```


## 📊 Тестовые сценарии

- Поиск в начале / середине / конце текста
- Отсутствующий шаблон
- Пустой шаблон
- Шаблон длиннее текста
- Худший случай (повторяющиеся символы)
- Большой текст (500000 байт)
- Русский текст (UTF-8)

## 📈 Результаты

Подробный анализ производительности и сводная таблица представлены в [`report.md`](report/output.txt).

## 👨‍🎓 Назначение

Проект выполнен в учебных целях для изучения:
- Алгоритмической сложности различных подходов к поиску подстроки
- Оптимизаций (префикс-функция, эвристики плохого символа и хорошего суффикса)
